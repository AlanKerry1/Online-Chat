## 🐳 Running  with Docker (🪟Windows)

> Requirements: Docker Compose

```bash
# Run from the project root
copy .env.docker.example .env.docker
```
You should set your env variables

```bash
# Run from the project root
docker-compose up -d
```

Once the container is running, open http://localhost:7070/ in your browser. Enter an account (use the random value in the bottom-left corner, no password required, as this is not an auth demo). Then open http://localhost:7070/ in a second window, enter another account and test
